# Real Estate Management System

## Project Description
The Real Estate Management System is a desktop application developed using **Java Swing** for the graphical user interface and **MySQL** for database management. The application is connected to a MySQL database through the **XAMPP server**, allowing users to securely store, retrieve, update, and manage real estate records.

This system is designed to simplify property management and make record handling more efficient and organized.

## Features
- Property record management
- Buyer and seller information management
- Property transaction tracking
- Add, update, delete, and search records
- Secure database connectivity using MySQL
- User-friendly Java Swing GUI
- Organized and structured data storage

## Technologies Used
- Java (Swing)
- MySQL
- XAMPP Server
- JDBC (Database Connectivity)
- GUI (Graphic User Interface)

## Project Modules
1. Property Management Module  
2. Buyer Management Module  
3. Seller Management Module  
4. Transaction Management Module  

## How It Works
- The application connects to a MySQL database using JDBC.
- Users can perform CRUD operations (Create, Read, Update, Delete).
- All data is stored securely in MySQL tables.
- Java Swing provides an interactive and easy-to-use interface.

## Setup Instructions
1. Install **XAMPP** and start Apache & MySQL.
2. Create a database in MySQL (e.g., `real_estate_db`).
3. Import the required SQL file (if provided).
4. Open the project in NetBeans or any Java IDE.
5. Add MySQL JDBC connector to the project.
6. Run the application.
